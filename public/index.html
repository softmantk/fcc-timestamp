Time Stamp Micro Service

Our Homw page : https://secret-island-36138.herokuapp.com/

Pass String as a parameter to https://secret-island-36138.herokuapp.com/string where the string should be a time stamp or a valid date.

  • If String is a valid time or unix time, the Output will be of the form as follows.
    https://secret-island-36138.herokuapp.com/20 oct 2015 30
    {"unix":6297565708800,"natural":"01-Dec-201531"}
  • If the string is invalid date, the out will be of the form
    https://secret-island-36138.herokuapp.com/20 oct 2015 30
    {"unix":null,"natural":null}
